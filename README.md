# OpusKitX
Prebuilt Opus framework for iOS, packaged for easy integration into your projects.
> ⚠️ This repository does NOT implement Opus. It provides a precompiled binary of the official Opus codec for convenient usage.

---

## ✨ Features

- 📦 Precompiled Opus framework (no need to build from source)
- ⚡ Easy integration into iOS projects
- 🔧 Suitable for CocoaPods / manual integration
- 🎧 Supports high-quality audio encoding and decoding via Opus

---

## 📦 Installation

### CocoaPods

```ruby
pod 'OpusKitX'
```

### Manual
1.	Download Opus.framework or Opus.xcframework
2.	Drag it into your Xcode project
3.	Add it to Frameworks, Libraries, and Embedded Content

   
## 🚀 Usage

### Objective-c
```Objective-c
#import <Opus/opus.h>
```

### Swift
```Swift
#import Opus
```

## 📚 About Opus

Opus is an open, royalty-free audio codec standardized by the IETF.

🔗 https://opus-codec.org/

## 🔄 Version

This build is based on Opus 1.6 (example).
